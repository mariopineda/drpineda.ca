Objectives
Program the game pong using Scratch
Teacher Instructions
  1. Go over the instructions with class before going to the computer room.
  2. Watch first half of video clip in class.
  3. Make sure each student has headphones.
  4. Hand out authentication credentials for Scratch web site te each student.
Student Instructions
  1. Before you start please read through all of the instructions (you do not have to watch the video clip just yet).
  2. Go to the Scratch web page and login using the user name and password given to you by your teacher.
  3. Create a new project and give it a name. Make sure the project name includes your own first name. For example, I would give my pong game project the following name: "Mario's game of pong".
  4. In this less on you will be creating your own version of the game pong using scratch. if you are not familiar with pong you can read more about it here:
    • Wikipedia
    • Pong Game
    • The story of pong

  5. Before starting to program watch the first half (up to time point 5:50) of the following YouTube video (use your headphones please). This part of the video explains how one could program pong in Scratch. Note that this is just one possible way you could program the game. You are not supposed to be copying exactly what the person in the video is doing, as a matter of fact, in our version of Scratch not everything will work the way it works in the video. Use the video for pointing you in the right direction and for general programming ideas. At this point do not worry about the second half of the video describing how to count points.

  6. Once you are ready to start programming these are the general steps you will be going through:
    1. Create and program the movements of the paddle sprites.
      • Make sure you use "sensible" keys for controlling the paddles, for example left paddle should be controlled by keys on the left side of the key board and the right paddle should be controlled by keys on the right side of the keyboard.
      • Note that the way the video describes how the paddles bounce of the edges might not work properly. Think about how you can fix this potential problem.
    2. Create and program the movements of the ball sprite.
      • Make sure the "Set costume centre" is set properly
      • Make sure the ball sprite starts in the centre.
    3. Try out your game and, once it is working properly, have it approved by your teacher.
    4. Have the ball sprite bounce of the edges and off the paddles.
    5. Once your teacher has approved your game you can either customize your games look to look like the classical pong (black background with white sprites and bouncing sounds) or you can watch the second half of the video and add counting points to your pong game.
